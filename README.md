# 5L-VSI-Fault-Dataset

This repository contains the simulated dataset for the thesis:
**"Detection of Open Switch Faults in Cascaded Multilevel Inverters Using Artificial Intelligence Techniques"**

---

## English Description
This dataset consists of phase current signals and processed feature matrices from a three-phase 5-level voltage source inverter (VSI).

- **File Contents:**
    - **Raw Data (.mat):** Original phase current signals for 16 different classes.
    - **Features_Data.mat:** The processed feature matrix (mean/rms ratios) used as input for Machine Learning models.
- **Load Stability:** The dataset includes tests under different R-L loads.
- **Technical Requirement:** **MATLAB** is required to open and process `.mat` files.

## Türkçe Açıklama
Bu depo, aşağıdaki yüksek lisans tezi için üretilen simüle edilmiş veri setini içermektedir:
**"Kaskat Bağlı Çok Seviyeli Eviricilerde Açık Anahtar Arızalarının Yapay Zekâ Teknikleri Kullanılarak Tespiti"**

- **Dosya İçerikleri:**
    - **Ham Veriler (.mat):** 16 farklı sınıf için kaydedilmiş orijinal faz akım sinyalleri.
    - **Features_Data.mat:** Makine öğrenmesi modellerine girdi olarak verilen, işlenmiş öznitelik matrisi (Ortalama/RMS oranları).

- **Teknik Gereksinim:** `.mat` dosyalarını görüntülemek ve işlemek için **MATLAB** yazılımı gereklidir.
